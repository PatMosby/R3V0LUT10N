String swp.bibjsf.presentation.BookForm.copySelected ( )

Copy the attributes of the selected book imported from Google to the book currently being added. If none is selected, but there is only one found book, that book is used for the copy. This method is an action called from the facelet.

Looks up the books matching the entered ISBN/ISSN at Google. Fills the data model 'foundBooks' with the retrieved books. If only one book is found, that books is entered in the edit form immediately. Merges all selected attributes from fromBook into toBook.
